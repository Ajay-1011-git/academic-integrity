\# VIT Academic Integrity Platform



A VIT-exclusive web platform for academic integrity in assignment submission and evaluation.



\## Features

\- Professor assignment creation with custom rubrics

\- Student submission with draft tracking

\- Plagiarism-aware evaluation system

\- Role-based access control (VIT email domains only)



\## Tech Stack

\- \*\*Frontend:\*\* React + Vite + Tailwind CSS

\- \*\*Backend:\*\* Node.js + Express

\- \*\*Database:\*\* Firebase Firestore

\- \*\*Authentication:\*\* Firebase Auth

\- \*\*Storage:\*\* Firebase Storage

\- \*\*Deployment:\*\* Vercel (Frontend) + Render (Backend)
